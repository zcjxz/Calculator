# Calculator
一个简单的计算器，可以一次性进行多则运算  
## 说明  
1. 为了使布局布局分布均匀，使用LinearLayout的weight进行分配，嵌套linearLayout，使布局能够适配各种尺寸的屏幕
2. 因为Java中float，double这些浮点型数据不能进行精细的计算，只能得出近似值，所以使用了BigDecimal这个类进行进行计算的处理
